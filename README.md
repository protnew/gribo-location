# GriboLocation #6620

Mushroom hunting GPS tracker with gamification: squats = mushrooms on the map.

## Tech Stack
- Frontend: Vite + Vanilla JS + Leaflet 1.9.4 + H3 + PWA (Workbox)
- Backend: FastAPI + SQLAlchemy + SQLite/PostGIS + Redis
- Mobile: Capacitor (Android)
- Tests: Playwright (E2E) + Vitest (unit)

## Quick Start

```bash
cd frontend && npm install && npm run dev
```
